# منصة المراسلات الذكية

منصة عربية مؤسسية لتوليد المراسلات الرسمية والتشغيلية لإدارة عمليات التدريب في جامعة نايف العربية للعلوم الأمنية.

## المزايا الرئيسية
- إدخال بيانات أسبوعية للدورات مع دعم الاستيراد من Excel.
- توليد بريد أسبوعي مخصص لعدة جهات.
- مكتبة سيناريوهات تشغيلية واعتمادية وتنسيقية.
- واجهة عربية RTL متجاوبة للجوال والتابلت وسطح المكتب.
- نسخ فوري للنصوص ومعاينة HTML للبريد.
- بنية جاهزة للتوسع نحو Outlook وMS Graph.

## التشغيل
```bash
npm install
npm run dev
```

## ملاحظات
- تم تجهيز تكامل Outlook كطبقة جاهزة للتوسعة وليس مفعلاً بالكامل في النسخة الأولى.
- التخزين في النسخة الحالية محلي عبر localStorage لتسريع الانطلاق.
