# منصة المراسلات الذكية

منصة ويب عربية مبنية بـ **Next.js + TypeScript + Tailwind CSS** لتسهيل إنشاء المراسلات الرسمية المتكررة لموظفي **إدارة عمليات التدريب** في **جامعة نايف العربية للعلوم الأمنية**.

## المزايا
- 16 قالب مراسلة موزعة على 4 مجموعات رئيسية
- نماذج ديناميكية حسب نوع الرسالة
- معاينة فورية للموضوع والنص
- نسخ الرسالة بنقرة واحدة
- استيراد بيانات الجداول من Excel
- دعم RTL كامل
- متجاوب مع الجوال والتابلت وسطح المكتب

## التشغيل المحلي
```bash
npm install
npm run dev
```

## البناء
```bash
npm run build
npm start
```

## المجلدات الرئيسية
- `app/` صفحات Next.js
- `components/` مكونات الواجهة
- `lib/` الأنواع والقوالب والثوابت والأدوات
- `hooks/` React hooks
- `public/` الأصول العامة
